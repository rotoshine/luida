## ② 설계 의사결정
- **요약 정책 주입**: `summarize: Fn(&[String])->Result<String>`로 트리 구조와 요약 생성을 분리 → LLM 없이 구조 단위테스트. 실사용은 `learning.reflect` 행위 주입(후속).
- **leaf 1개 = 루트**: 요약 노드 불필요. fanout≥2 강제(무한루프 방지).
- **원본은 vault(.md), 요약은 DB**: leaf.path가 원본 가리킴. reflect/plan은 `roots()`/상위 레벨만 읽어 토큰 절감(spec §6.1).
- **문단 경계 보존 청크화**: `\n\n` 단위로 누적, 예산 초과 시 flush. 단일 거대 문단은 단독 청크(추가 분할은 후속).
